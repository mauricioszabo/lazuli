# Lazuli
Pulsar package for interactive development for the Ruby language.

Kinda, but not really, a fork of [Chlorine](https://gitlab.com/clj-editors/atom-chlorine) but for the Ruby language, obeying the limitations and capabilities of Ruby itself.

For the full source and README, please check [https://gitlab.com/clj-editors/lazuli/](https://gitlab.com/clj-editors/lazuli/)
